# Day 02 - Python OS Module

## Objective

Learn how Python communicates with the Operating System and navigate the file system.

## Topics Covered

- Operating System
- os Module
- Current Working Directory
- Files and Directories
- Paths

## Python Functions

- os.getcwd()
- os.listdir()
- os.path.isfile()
- os.path.isdir()
- os.path.abspath()

## Practical Project

Built a Directory Inspector that lists all files and folders in the current directory.

## Reflection

Today I learned that automation begins with understanding the file system. Before automating tasks, software must first discover and understand the files it needs to process.
