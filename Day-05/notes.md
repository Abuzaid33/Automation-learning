What is a dictionary in Python?
A dictionary is a datatype in python which is used to store multiple item in form of key value pair.
How is a dictionary different from a list?
A list store a multiple single values but dictionary store them in pairs also the list items can be fetched by index number while dictionary items can be fetched by keys
What is a key?
key is the first in pair in dictionary item which is used to fetch the value
What is a value?
the value is the second in the pair which is fetched by the key 
Why are dictionaries useful in automation?
when we use to store items in form of key values pair we used dictionaries for example in automation we can store logs and also login data like password and username
